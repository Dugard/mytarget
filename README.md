MyTarget API Client for PHP.
==============================

Простой и удобный PHP-клиент [MyTarget Api](https://target.my.com/adv/api-marketing/).

## Требования
 * PHP 7.0 и выше
 
## Установка  
В файле `composer.json`:
```php
{
    ...
    "require": {
        ...
        "kradwhite/mytarget-api-client": "*"
    }
    ...
}
```
